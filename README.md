# 🧠 Repositório de Exercícios JavaScript

exercicios praticados pelo curso em video com o objetivo de aprender e compreender os conceitos de JavaScript aplicando de forma pratica em pequenas páginas 

Este repositório contém exercícios práticos de JavaScript, desenvolvidos como parte do meu aprendizado em desenvolvimento web. Aqui você vai encontrar desafios de lógica, manipulação do DOM, funções, loops, arrays, objetos e muito mais.

## 🧪 Tópicos abordados

- ✅ Variáveis e tipos de dados
- ✅ Condicionais (`if`, `switch`)
- ✅ Laços de repetição (`for`, `while`)
- ✅ Arrays e objetos
- ✅ Funções e callbacks
- ✅ Manipulação de DOM (básico)
